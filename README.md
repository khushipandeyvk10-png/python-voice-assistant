
# Personal Python Voice Assistant

A lightweight, local Python-based voice assistant designed to process real-time audio commands and respond with active desktop automation and text-to-speech feedback.

## 🛠️ Features Implemented
- **Speech Recognition:** Converts live microphone audio into string inputs natively.
- **Text-to-Speech Engine:** Provides vocal feedback directly back to the user.
- **Automated Desktop Tasks:** Launches web browser searches and calculates date/time dynamically.
- **Graceful Error Catching:** Handles ambient noise or unrecognized syntax smoothly.
